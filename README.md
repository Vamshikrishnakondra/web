# 🍽️ Swiggy Business Analytics & Revenue Forecasting Dashboard

An interactive **Streamlit Dashboard** built using **Python, Pandas, Plotly, Matplotlib, and Machine Learning** to analyze Swiggy sales data, discover business insights, and forecast future revenue.

---

## 📌 Features

- 📊 Interactive Dashboard
- 📈 KPI Metrics
- 🏙 Revenue Analysis by City & State
- 🍴 Top Restaurants Analysis
- 🍕 Top Dishes Analysis
- 🥗 Veg vs Non-Veg Revenue Comparison
- ⭐ Ratings Analysis
- 📅 Daily & Monthly Revenue Trends
- 🔥 Weekend vs Weekday Performance
- 📉 Correlation Analysis
- 🤖 Revenue Forecasting using Linear Regression
- 📥 Download Processed Dataset
- 📱 Responsive Streamlit UI

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Swiggy-Analytics.git
```

Move to project folder

```bash
cd Swiggy-Analytics
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run Streamlit

```bash
streamlit run app.py
```
